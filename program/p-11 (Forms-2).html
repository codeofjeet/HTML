Commonly Used Input Types in HTML Forms

* <input type="password"> :- is used to specify the password field of the input tag. Password should be served over the HTTPS pages because it includes the sensitive information of the user.



========================================================

* <input type="submit"> :- is used to define a button that submits form data to the server when clicked. It is commonly used in various forms, such as email forms, contact forms, or login forms, to send the data for processing.





Attributes

A. value:- Specifies the text displayed on the submit button.

========================================================

B. name :- Defines a name for the submit button, useful for identifying the button in the form data.
