Images tag

The HTML "img" tag is a handy tool for embedding images in web pages by linking to them. It sets up a space for the image, which is defined by attributes such as src, width, height, and alt, and it doesn’t need a closing tag.
You can insert images into a webpage in two ways:
1. By providing a full path or URL to access an image file on the internet.
2. By using a file path that’s relative to where the current web page file is located.
